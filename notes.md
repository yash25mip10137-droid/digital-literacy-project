## Part A – Coding Practice: HackerRank

**Platform Used:** HackerRank (hackerrank.com)

**Challenge Completed:** Solve Me First (Introduction Domain)

**Description:**
The "Solve Me First" challenge is HackerRank's introductory problem. It asks the user to write a function that takes two integers as input and returns their sum. This challenge is designed to help new users understand how HackerRank's input/output system works.

**Solution (Python):**
```python

    return a + b

a = int(input())
b = int(input())
print(solveMeFirst(a, b))
```

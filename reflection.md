## Reflection (150–200 words)

For this task, I set up profiles on three platforms: GitHub, LinkedIn, and Kaggle. Each platform serves a distinct purpose in a student's academic and professional journey.

GitHub is the world's largest code hosting platform, used by developers and students to store projects, collaborate on code, and contribute to open-source software. I created a profile README that introduces me and states my learning goals — this acts as my coding identity card.

LinkedIn is a professional networking platform used by recruiters, companies, and professionals worldwide. I updated my Education section with my current degree and expected graduation year. Even as a first-year student, having a LinkedIn profile means I am ready to start building connections early.

Kaggle is a platform for data science and machine learning enthusiasts. It hosts competitions, datasets, and notebook environments. I signed up here because CSE students are increasingly expected to have data skills.

Over the next four years, I plan to use GitHub to host all my projects, LinkedIn to connect with industry professionals and apply for internships, and Kaggle to practice data skills and participate in competitions.
